# flutter_simulador_bitola

Calculadora de Bitola (mm²)

Projeto desenvolvido em Flutter com o objetivo de praticar conceitos iniciais de desenvolvimento mobile, criando uma aplicação para cálculo da bitola de cabos elétricos em instalações residenciais.

- Funcionalidades
Cálculo da bitola para redes 110V
Cálculo da bitola para redes 220V
Entrada de dados personalizados:
Distância (metros)
Corrente elétrica (Ampères)
Exibição dos resultados de forma clara e objetiva
Interface simples e intuitiva
